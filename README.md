# Financial Performance Dashboard – Power BI

## Project Overview
This project consists of the development of an interactive financial dashboard in Power BI to analyze the financial performance of a company over a three-year period.

The dashboard focuses on key financial indicators such as revenue, expenses, profit and profit margin. It allows users to explore the data dynamically through filters by year, country, brand and customer, enabling analysis from different business perspectives.

The objective of this project is to identify the most profitable countries, brands and customer segments, as well as to analyze financial performance trends over time to support data-driven decision making.

The dataset used in this project is simulated and was used for learning and dashboard development purposes.

---

## Tools Used
- Power BI
- DAX
- Excel

---

## Dashboard Features
- KPI cards:
  - Total Revenue
  - Total Expenses
  - Total Profit
  - Profit Margin

- Filters (Slicers):
  - Year
  - Country
  - Brand
  - Customer

- Visualizations:
  - Profit and profit margin by country
  - Profit margin by brand
  - Profit by customer
  - Monthly profit and margin trend
  - Financial summary table (Revenue, Expenses, Profit)

---

## Analysis Overview
The dashboard enables the analysis of financial performance across multiple dimensions, allowing users to compare profitability by country, brand and customer, as well as evaluate trends over time.

The analysis shows that profitability varies across countries and years, meaning business performance is not constant over time. Some brands generate higher profit margins than others, and certain customer segments are more profitable even if they do not generate the highest revenue.

This type of analysis helps identify the most profitable areas of the business and supports strategic decision making.
